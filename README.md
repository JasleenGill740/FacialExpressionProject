# FacialExpressionProject
#### A deep learning-based project that detects human facial expressions (Angry, Happy, Sad, Neutral, Surprise) from images using CNN.

## Dataset 
#### Due to the large size of the dataset, it is available here:
https://drive.google.com/drive/folders/1psixAQpxsnIPqiT5Nnu0H4_u2Ot832Dc

## How to use the dataset
#### 1.Open the dataset link.
#### 2.Download the Data folder.
#### 3.Place the Data folder in the project directory.

## How to Test
#### Place your test images inside the Test_pictures/ folder.
#### Open Project.ipynb.
#### Run all cells.
#### The model will predict the facial expression for the test images.
